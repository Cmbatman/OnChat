# OnChat Premium Design System

## Core Aesthetic
A sophisticated, conversation-focused interface that balances brand energy (Primary Orange) with professional depth.

## Components Style Guide

### 1. Buttons
- **Primary**: Solid #FF8132, White text, 8px radius.
- **Secondary**: Solid #00B2E0, White text, 8px radius.
- **Interaction**: Scale 1.02 on hover.

### 2. Cards & Containers
- **Main Shell**: 16px radius, 1px border (#333333).
- **Sub-cards**: 12px radius.

### 3. Identity System (Genders)
- **Avatars**: Circular with 2px accent ring.
- **Gender Tiles**: 120x120px interactive blocks.

### 4. Form Fields
- **Inputs**: 8px radius, #1F1F1F background (Dark).
- **Focus**: 2px Brand Primary border.

### 5. Overlays
- **Modals**: 16px radius, centered, 8px backdrop blur.

### 6. Scrollbars
- **Thumb**: #D1D1D1, 8px width.
